Fabric.js 是一个简单且强大的
Javascript HTML5 canvas 库

Fabric 在canvas中提供了 可操作对象模型

Fabric 也有 SVG-to-canvas (和 canvas-to-SVG) 渲染器

使用 Fabric.js, 你可以在canvas中创建填充一个对象; 比如 几何图形
or 由成百上千个简单路径组成的复杂图形
or 好看的 图片
你可以添加 文字 并且能够动态的修改 大小, 对齐方式, 字体, 以及其他属性
你可以对任何形状进行 倾斜
并且 在图片上 添加滤镜
还有内置的 动作系统
你可以将多个对象分组, 在同时修改它们
你可以给任何对象增加阴影. 或者让他变得 可拖拽 试试这个星星 →
对象可以轻易的向不同方向翻转. 或者 锁定 运行方向, 缩放等.
← 试试这个猫.
还有一个强大的 事件 系统
你可以 随便裁剪 图形
或者用图案 填充图形
当然, 可以使用 自由画笔 创造任何你想要的,
同时也支持橡皮擦
画布 可以序列化 为 JSON 或者 SVG, 并且 支持随时复原
